
## React Redux SSR Demo
ReactRouter4 + Express + Axios + Redux + Scss + Image loader

## For Development
### make sure npm is installed
1. `npm install`
2. `npm run dev`
3. `open http://localhost:3000`

### For Production
1. `npm install`
2. `npm run build`
3. `npm run start`
4. `open http://localhost:3000`
